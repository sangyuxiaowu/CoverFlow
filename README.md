CoverFlow - 专业 SVG 文章封面设计器
=====================================

面向多平台文章封面图的 Web 设计工具。基于 SVG 资源库与图层系统，提供可配置画布、快速排版、背景生成与项目保存导出能力，适合日常内容运营与设计协作。

核心功能
--------

### 画布系统

- 多比例画布：内置微信文章封面比例（2.35:1、1:1）
- 扩展比例：可配置添加其他平台比例（16:9、4:3、3:2 等）
- 画布控制：缩放、平移、网格对齐辅助线
- 实时预览：在不同比例间快速切换检查效果

### SVG 资源管理

- 分类资源库：图标、装饰、背景元素、文字框等
- 资源上传：支持单个 / 批量 SVG 上传
- SVG 解析：自动解析内部结构并支持元素拆分
- 搜索与过滤：按分类、标签、颜色筛选
- 资源预览：缩略图展示，悬停查看详细信息

### 图层编辑系统

- 图层面板：树形结构展示全部图层
- 图层操作：
	- 位置调整（XY 拖拽 / 精确数值输入）
	- 缩放（等比 / 非等比，拖拽 / 数值控制）
	- 旋转（自由旋转 / 15 度步进）
	- 顺序调整（上移 / 下移 / 置顶 / 置底）
	- 显示 / 隐藏 / 锁定
- SVG 拆分与组合：
	- 复合 SVG 拆分为独立元素图层
	- 多元素组合为新图层组
	- 支持进入组内编辑

### 背景系统

- HTML/CSS 背景生成器：
	- 纯色填充
	- 线性 / 径向渐变
	- 图案填充（条纹、网格、点阵等）
	- CSS 代码直接输入
- 背景图片：上传图片作为背景
- 预置背景模板：常用配色方案与图案

### 导出与分享

- 图片导出：PNG、JPG
- 多分辨率导出：1x、2x、3x
- 批量导出不同比例版本
- 项目文件格式：JSON 保存项目状态
	- 图层信息、位置、样式
	- SVG 资源引用信息

### 项目管理

- 本地自动保存
- 历史版本：撤销 / 重做
- 模板保存：将设计保存为模板
- 项目导入 / 导出：JSON 文件导入恢复编辑状态

技术栈
------

- React 19 + TypeScript
- Vite 6
- html-to-image（导出）

本地开发
--------

前置要求：Node.js 18+（推荐 20+）

```bash
npm install
npm run dev
```

构建与预览：

```bash
npm run build
npm run preview
```

目录结构
--------

```
.
├─ components/         # UI 组件与面板
├─ utils/              # 工具函数
├─ App.tsx             # 应用入口
├─ constants.ts        # 常量与默认配置
├─ types.ts            # 类型定义
└─ translations.ts     # 文案与多语言
```
